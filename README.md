# Note

 インスタグラムアカウントのパスワードが短いと、安全性（bot）チェックが入るのでうまく動作しない場合がある。
 プログラム起動時、別Windowを開くとうまく動作しない場合があるのでプログラムが動作したらPCを触らない。
